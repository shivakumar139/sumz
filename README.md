# Development
1. Clone the repo into a public GitHub repository

```sh
   git clone https://github.com/shivakumar139/sumz/
```

2. Go to the project folder

```sh
   cd sumz
```

3. Install packages with yarn

```sh
   yarn
```

4. Set up your ```.env``` file


5. Run (in development mode)

```sh
   yarn dev
```

6. Visit

```sh
   http://localhost:5173
```
